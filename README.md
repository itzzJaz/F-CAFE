# ☕ T-Cafe Timer App

T-Cafe is a fun cafe-themed timer web app where you act like a customer placing orders. Each food item has a timer, and you can also set custom timers manually.

## 🎯 Features

- 🧑‍💻 Enter your name and get a personalized experience
- 🍔 Quick order buttons for cafe items (coffee, pizza, pasta, etc.)
- ⏱️ Start / Stop / Reset timer controls
- 🕒 Custom timer input (hours, minutes, seconds)
- 🍽️ Each item has its own cooking/waiting time
- 🔊 Sound plays when timer is done
- 💬 Fun cafe-style UI with speech bubble interaction

## 🍕 Available Items

- ☕ Coffee (10 min)
- 🍝 Pasta (25 min)
- 🥐 Croissant (30 min)
- 🌯 Sandwich (45 min)
- 🍛 Rice Bowl (60 min)
- 🍔 Burger (15 min)
- 🍕 Pizza (20 min)
- 🍩 Donut (12 min)
- 🍨 Ice Cream (8 min)
- 🍜 Noodles (35 min)
- 🍟 Fries (18 min)
- 🥩 Steak (40 min)
- 🫖 Tea (5 min)
- 🍲 Ramen (50 min)
- 🍰 Cake (14 min)

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript

## 🚀 How to Run Locally 

1. Download or clone this repository
2. Open `index.html` in your browser
3. Start ordering and use the timer!
